Description: This a simple jumia phone service to access and validate phone numbers for many countries.

------------------------------------------------------------------------
Back end Application:
------------------------------------------------------------------------
Clone the application source code:
-----------------------------------
use this URL: https://github.com/hanyabdelbaqy/jumia-phone-service-back-end.git

Building the application you need:
-----------------------------------
JDK 1.8
Maven 3

Running the application:
-----------------------------------
to run this application flow the below methods.

Method #1
using the attached jar, you can run this command "java -jar jumia-phone-service-1.0.jar".

Method #2
you can use any IDE like Eclipse, STS or IntelliJ, then clean, build and run.

Accessing the application:
-----------------------------------
Port: 9595

You can access this URL "http://localhost:9595/swagger-ui/index.html" to check Swagger API Documentation.
You can access this URL "http://localhost:9595/jumia/services/customers" to get all phone numbers.

------------------------------------------------------------------------
Front end Application:
------------------------------------------------------------------------
Clone the application source code:
-----------------------------------
use this URL: https://github.com/hanyabdelbaqy/jumia-phone-service-front-end.git


Running the application:
-----------------------------------
to run this application flow the below methods.

Method #1
using the attached dist folder, you can use NGINX server to deploy it.

Method #2
you can use any IDE like Webstorm to clean, build and run.

Accessing the application:
-----------------------------------
Port: 4200

You can access this URL "http://localhost:4200/customers" to get all phone numbers and use user filter inputs.

Note: for user filter input, you can entry country name ex: Morocco, Cameroon, Ethiopia for Country input and entry true or false for valid input
